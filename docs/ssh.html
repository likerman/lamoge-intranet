Guía de acceso por SSH

Este documento resume el procedimiento recomendado para solicitar acceso, cargar claves públicas y conectarse a los servidores del laboratorio de forma segura.

Requisitos previos

  • Contar con autorización previa para el servidor correspondiente.
  • Disponer de una clave pública registrada por administración.
  • Conocer el nombre de usuario asignado y el hostname de destino.

Comando de conexión

ssh usuario@servidor

Agregue aquí variantes con puerto específico, proxy o archivos de configuración si fueran necesarias.

Seguridad operativa

  • No comparta credenciales ni claves privadas.
  • Describa el procedimiento para rotación de claves y revocación de accesos.
  • Documente restricciones de IP, VPN o segmentación de red si aplican.
